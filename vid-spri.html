Vid Spri

VidSpri fue creado pensando en los desarrolladores novatos o aquellos que quieren crear sus videojuegos 2D pero que no tienen experiencia dibujando cada fotograma.

A muchos les ha ocurrido acudir a la IA para generar fotogramas para sus personajes, pero la IA tampoco puede hacer bien este trabajo; no sabe cómo hacerlo hasta que nos dimos cuenta de que la IA, al generar video, lo hace mucho mejor que cuando genera una simple imagen con varios sprites.

Se nos ocurrió hacer una herramienta que convierte videos a hojas de sprites. Nuestra herramienta saca fotogramas de videos y los convierte en una hoja de sprites o genera un video para sacarle fotogramas, logrando un mejor resultado.

Para hacerlo más completo, hemos hecho que pueda generar sonido para tus videojuegos.

Tecnologías y Librerías de Código Abierto

Python

El lenguaje de programación que impulsa todo nuestro backend.

Docker

La plataforma de contenedores que nos permite desplegar el servidor de forma fiable.

FastAPI

Un moderno y rápido framework web para construir nuestras APIs.

Uvicorn

Un servidor ASGI ultrarrápido, utilizado para ejecutar la aplicación de backend.

rembg

La potente librería utilizada para eliminar el fondo de las imágenes de forma inteligente.

Pillow

Una librería fundamental para la manipulación de imágenes en Python.

ONNX Runtime

El motor de inferencia de alto rendimiento que potencia el modelo de rembg.

Python-Multipart

Librería esencial para la gestión de subida de archivos en el servidor.

SQLite

El motor de base de datos utilizado para gestionar la cola de procesamiento y los códigos de prioridad.

Google Translate

El servicio que potencia la funcionalidad de traducción en la aplicación.

Feather Icons

La librería de iconos de código abierto que proporciona los iconos de la interfaz de usuario.

ImageCrop

Una librería de JavaScript para la funcionalidad de recorte de imágenes en el frontend.
