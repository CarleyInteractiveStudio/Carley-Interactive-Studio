Documentación de Creative Engine

Introducción

Bienvenido a la documentación de Creative Engine. Aquí encontrarás todo lo que necesitas para empezar a crear tus propios videojuegos directamente en la web.

Primeros Pasos
1. Crear un Nuevo Proyecto

Para iniciar, haz clic en "Archivo" > "Nuevo Proyecto". Se te pedirá que elijas una carpeta en tu ordenador donde se guardarán todos los archivos de tu proyecto.

2. La Interfaz

La interfaz se divide en varias ventanas: la Jerarquía (lista de objetos en tu escena), la Escena (vista visual de tu juego), el Inspector (propiedades del objeto seleccionado), y el Navegador de Archivos (tus assets).

Creative Engine Scripting
Conceptos Básicos

Creative Engine Scripting (CES) es un lenguaje basado en JavaScript diseñado para ser fácil de aprender. Cada script es un componente que puedes añadir a un objeto del juego.

Funciones principales:

  • iniciar(): Se ejecuta una vez cuando el juego comienza.
  • actualizar(): Se ejecuta en cada fotograma del juego.
Ejemplo: Mover un Personaje

Crea un script llamado "MovimientoPersonaje" y añade este código:


publico velocidad = 5;

funcion actualizar() {
    si (Entrada.teclaPresionada("Derecha")) {
        este.transformar.posicion.x += velocidad * Tiempo.deltaTiempo;
    }
    si (Entrada.teclaPresionada("Izquierda")) {
        este.transformar.posicion.x -= velocidad * Tiempo.deltaTiempo;
    }
}

Arrastra este script a tu objeto "Personaje" en la jerarquía. ¡Ahora puedes moverlo con las flechas!
